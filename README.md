# SmartFlux

Landing page estática da SmartFlux para diagnóstico, implantação e operação Kommo.

## Estrutura

- `index.html` — marcação principal da landing page.
- `assets/css/styles.css` — estilos extraídos do HTML original.
- `assets/js/main.js` — interações, animações e simulador de preços.
- `preview.html` — arquivo original enviado, mantido para compatibilidade e para evitar conflitos de merge.
- `logo.png` — imagem enviada junto com a página.

## Como visualizar localmente

```bash
python3 -m http.server 4173
```

Depois acesse `http://127.0.0.1:4173/` no navegador.
